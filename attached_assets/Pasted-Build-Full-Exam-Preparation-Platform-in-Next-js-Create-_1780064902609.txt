Build Full Exam Preparation Platform in Next.js

Create a modern, production-ready, mobile-first exam preparation platform using Next.js App Router, TypeScript, Tailwind CSS, shadcn/ui, Redux Toolkit, and TanStack React Query.

The UI design should look premium, clean, minimal, modern, and inspired by educational apps like Exam Explorer, PhysicsWallah, Adda247, and Testbook, with soft shadows, rounded cards, smooth spacing, and a purple-white gradient theme.

The website must be highly scalable, SEO optimized, API-driven, responsive on all devices, and use reusable modular architecture. Avoid static hardcoded content as much as possible.
